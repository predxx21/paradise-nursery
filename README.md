# Paradise Nursery - Plant Shopping Application

## Project Overview
Paradise Nursery is a dynamic web application for an online plant shop that allows users to browse and purchase houseplants. The application provides an intuitive shopping experience with features like product categorization, shopping cart management, and real-time price calculations.

## Features
- Browse plants by categories (Indoor, Outdoor, Succulents)
- View plant details including images, names, descriptions, and prices
- Add plants to shopping cart with quantity management
- Real-time cart total calculation
- Responsive navigation with cart item counter
- Persistent cart state using Redux

## Technologies Used
- React.js
- Redux Toolkit for state management
- React Router for navigation
- CSS3 for styling

## Setup Instructions
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the development server
4. Open http://localhost:3000 to view the application

## Project Structure
- `/src/components` - React components
- `/src/features/cart` - Redux slice for cart management
- `/public/images` - Plant images and assets
